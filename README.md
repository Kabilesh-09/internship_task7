## Objective
Learn to detect risky browser extensions and remove them for improved cybersecurity.

## Steps Performed
1. Opened browser extensions page (chrome://extensions/)
2. Reviewed each extension's permissions, developer, and usage
3. Flagged suspicious extensions based on:
   - Unknown developer
   - Excessive permissions
   - No reviews/low rating
   - Not installed intentionally
4. Removed unsafe extensions
5. Restarted browser
6. Documented actions and findings

## Table: Suspicious Extensions (Example)
| Extension | Status | Reason |
|---|---|---|
Example: UltraFree VPN | Removed | Unknown vendor, collects browsing data |
Example: PDFConverterFree | Removed | Modifies search engine, privacy concerns |

## Key Learning
- Browser extensions can access sensitive data (browsing activity, cookies, passwords)
- Only install extensions from trusted developers
- Always review permissions before installing
- Remove unused extensions regularly
